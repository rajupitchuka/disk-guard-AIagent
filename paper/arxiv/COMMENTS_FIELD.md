## What goes in this field

The Comments field is shown on the public abstract page (e.g.,
`arxiv.org/abs/2605.XXXXX`) right under the abstract. Standard
contents:

- Page count and figure count
- Conference / journal venue if accepted (after acceptance — leave
  blank for the initial submission)
- A code repository URL if the paper accompanies a public
  implementation (this is YOUR case; include it)
- Notes about the paper's relationship to other work ("Extended
  version of a paper accepted at...", "v2 incorporates reviewer
  feedback...")

Keep it terse. One short sentence is the norm.
